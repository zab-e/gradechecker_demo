Student Grade Calculator (C)

This is a simple C program that lets you enter a score and tells you your grade.

Enter a score between 0 and 100

It will check if the score is valid

Then it prints your grade (A, B, C, D, or F)

How to Run
gcc grade_calculator.c -o grade_calculator
./grade_calculator


Just type your score when asked. If the score is out of range, it will say Incorrect score.
